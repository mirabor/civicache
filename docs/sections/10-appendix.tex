\section{Supplementary Figures}\label{sec:appendix}

This appendix provides the full figure set for reproducibility-minded
readers. Main-body figures (Figures~\ref{fig:opening}, \ref{fig:workload_viz},
\ref{fig:alpha_sens}, \ref{fig:shards_overlay}, and the three
\S\ref{sec:ablations} figures) carry the paper's argument; the figures below
expand the same data with per-workload detail, duplicate views used for
different purposes in the main text, and the per-policy delta breakdowns
that did not fit in the main body. All ten figures regenerate from a clean
checkout via \texttt{make run-sweep} followed by \texttt{make plots
WORKLOAD=congress} / \texttt{WORKLOAD=court} / \texttt{WORKLOAD=shards\_large}
and \texttt{python3 scripts/compare\_workloads.py}.
